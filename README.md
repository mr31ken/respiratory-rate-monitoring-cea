# Economic Evaluation of Automated Respiratory Rate Monitoring in General Hospital Wards

A reproducible, deterministic cost-minimization and break-even modeling study comparing manual respiratory rate (RR) counting with six automated monitoring scenarios across four national healthcare systems (Japan, UK, Australia, USA).

This repository contains all data, analysis code, and documentation accompanying the manuscript submitted to the *Journal of Clinical Monitoring and Computing*.

---

## Quick Start: Reproduce All Results

Requirements: Python ≥ 3.9.

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Regenerate all figures and tables from source data
cd analysis
python 05_generate_all.py
```

This regenerates every figure (`figures/fig0_*.png` through `fig5_*.png`) and every table (`tables/table1_*.csv` through `table6_*.csv`) deterministically from `data/`.

---

## Adapt the Model to Your Institution

All cost and clinical parameters are stored as plain CSV in `data/cost_parameters.csv`. To run the model with local data:

1. Edit values in `data/cost_parameters.csv` (each row contains a value, source citation, and "Published" / "Assumption" classification).
2. Re-run `python analysis/05_generate_all.py`.
3. Inspect `tables/table4_breakeven_analysis.csv` and the regenerated figures.

The data dictionary (`data/DATA_DICTIONARY.md`) defines every variable.

## Data Provenance

- Input CSVs in `data/` cite primary sources (NICE MIBs, FDA 510(k) summaries, peer-reviewed clinical validation studies, MHLW/NHS/AIHW/CMS government statistics, manufacturer disclosures).
- Parameters are explicitly classified as **Published** (cited source) or **Assumption** (model parameter requiring sensitivity testing); see Supplementary S2.
- All values were independently verified against primary sources; the verification log is in Supplementary S1.
